# Mobile APK Security Analysis
Static security analysis of popular Android applications using MobSF, VirusTotal, and Exodus Privacy. Reports include vulnerability findings mapped to OWASP Mobile Top 10 and OWASP MASVS, CVSS-aligned severity classification, and remediation recommendations.
